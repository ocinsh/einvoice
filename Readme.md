# einvoice — Archived

> **This repository has been archived and split into several parts.**

The original goal of this project was to build a **free invoicing program** that
supported the Italian SDI (Sistema di Interscambio) rules and could forecast the
taxes due.

That goal has been compacted into three macro repositories:

## [mindmain/eattura](https://github.com/mindmain/eattura)
The server/desktop application to manage invoices and send them through PEC
(certified email) with synchronization.

## [mindmain/eattura-proto](https://github.com/mindmain/eattura-proto)
Definitions and documentation of the electronic invoice in Protobuf, with
associated docs in Italian.

## [ocinsh/taxer](https://github.com/ocinsh/taxer)
Library and CLI utility for forecasting taxes in the Italian system (still
expanding, due to the strong complexity component of the Italian system).

---

The code in this repository is kept for reference only and is no longer
maintained. Please refer to the repositories listed above for active
development.
